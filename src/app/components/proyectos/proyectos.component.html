PROYECTOS REALIZADOS

PROGRAMA EN C++ DE GESTION COMERCIAL

Este proyecto fue desarrollado como una prueba para un negocio con el objetivo de crear un sistema de gestión eficiente que permita llevar un control completo del inventario, usuarios y stock. Para lograr esto, se diseñó y desarrolló un sistema de gestión con una interfaz intuitiva y fácil de usar que permite a los usuarios administrar y actualizar el inventario, agregar nuevos productos, eliminar los que ya no se venden y realizar seguimiento del stock disponible. Además, se creó un sistema de usuarios con diferentes roles y permisos para garantizar la seguridad y privacidad de la información. Este proyecto se enfocó en ofrecer una solución de software que sea altamente personalizable y escalable para adaptarse a las necesidades de cualquier negocio y mejorar sus operaciones diarias
